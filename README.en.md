# 家庭医生

#### Description
“家庭医生”小程序是为了方便您根据症状进行自我评估。本程序是在军事医学科学出版社《家庭保健百科全书》的基础上，按照情景对话式风格设计。 本程序的亮点再有一种轻量级的问答逻辑方案。 已经发布微信小程序。
